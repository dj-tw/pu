# pu

For Census data

Install with install script

```
./install.sh
```

Activate virtual env (will need to do whenever you open a new shell)

```
source venv/bin/activate
```

Run tests

```
./run_tests.sh
```

Open ipython

```
ipython
```

```
from pu.read_pandas import read_pandas
data = read_pandas()
```
